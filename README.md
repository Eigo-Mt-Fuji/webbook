# webbook

## 開発環境

* JDK 8
* Tomcat 8
* Maven 3.x
* PostgreSQL 9.6

## Local

* Required system properties

```
-Ddb_host="/*DBホスト名*/" -Ddb_port="/*DBポート番号*/" -Ddb_name="/*DB名*/" -Ddb_user="/*DBユーザ名*/" -Ddb_password="/*DBパスワード*/"
```

* [ログインURL](http://localhost:8080/webbook/session)
